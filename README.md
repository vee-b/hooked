## üöÄ Getting Started

Follow these steps to run the project locally.

### 1. Clone this repository

```bash
git clone https://github.com/vee-b/hooked.git 
cd hooked
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server (Tauri)

```bash
npm run tauri dev
```
